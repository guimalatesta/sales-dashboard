## Setup

### Reqs

- Node.js 18+
- MongoDB 7+
- Redis 7+ (optional)
- Git

### Install

```bash
git clone https://github.com/seu-usuario/sales-dashboard.git
cd sales-dashboard

# Backend
cd backend
npm install
cp .env.example .env
npm run seed
npm run dev

# Frontend
cd ../frontend
npm install
npm run dev
```
